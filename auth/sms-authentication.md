# SMS Authentication

## Test Case ID
TC-AUTH-001

## Title
Verify successful authentication via SMS code

## Priority
High

## Preconditions

- Registered user exists
- User has valid mobile number
- SMS service available

---

## Steps

### 1
Open authentication screen

**Expected Result**

Authentication modal is displayed

---

### 2
Enter valid phone number

**Expected Result**

Phone number accepted

---

### 3
Tap "Continue"

**Expected Result**

SMS code input screen appears

---

### 4
Enter valid verification code

**Expected Result**

User is authenticated successfully

---

### 5
Navigate to profile

**Expected Result**

User profile page opens
